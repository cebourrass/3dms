# 3DMS Pro Analyzer - C# / WPF

Cette application est une refonte moderne de l'interface d'analyse de données 3DMS.

## 🚀 Comment Compiler et Exécuter

1.  **Ouvrez Visual Studio 2022**.
2.  **Ouvrez le dossier** `c:\dev\3DMS-CED`.
3.  Visual Studio devrait détecter le fichier `.csproj`.
4.  Attendez que les **packages NuGet** soient restaurés (ModernWpf, LiveCharts2).
5.  Appuyez sur **F5** ou cliquez sur le bouton "Play" pour lancer l'application.

## 📦 Stack Technique
- **Framework** : .NET 8.0 (WPF)
- **UI** : ModernWpf (Look & Feel Windows 11)
- **Graphiques** : LiveCharts2 (Haute performance pour la télémétrie)
